Mustache.js - All Templates, No Logic

  • Designed to make you hate inserting dynamic content less
  • Entirely logicless aka make sure your data is in order

	var obj = {
		name: 'Corbin Dallas',
		pass: {
			type: multi,
			secondary_user: 'Leeloo Dallas'
		}
		elements: [{type: earth}, {type: fire},
		{type: water}, {type: wind}, {type:leeloo}]
	}
						

	<section>
		<h3 class="name">{{name}}</h3>
		{{#pass}}
		<h3>{{type}}pass. Also authorized for {{secondary_user}}</h3>
		{{/pass}}
		<h3>The following elements have been collected:</h3>
		<ul>
		{{#elements}}
			<li>{{type}}</li>
		{{/elements}}
		</ul>
	</section>

Neat, but limitations

  • Logicless templates makes for a clean solution...
  • ...but can result in a lot of object preprocessing
  • No real conditional or loop helpers
  • Can't easily access current index in arrays

Could be more… hipstery.

Handlebars.js

  • Extension of Mustache.js
  • Can import and compile mustache.js templates
  • Crazy extensible - add your own helpers
  • Provides a slew of conditional and looping helpers

	Handlebars.registerHelper('list', function(items, options) {
	  var out = "<ul>";

	  for(var i=0, l=items.length; i<l; i++) {
	    out = out + "<li>" + options.fn(items[i]) + "</li>";
	  }

	  return out + "</ul>";
	});
